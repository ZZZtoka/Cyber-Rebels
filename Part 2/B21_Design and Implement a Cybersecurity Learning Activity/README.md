## B21_Design and Implement a Cybersecurity Learning Activity

## Description
For this activity, I designed and implemented a cybersecurity learning activity focused on phishing awareness for university students using Google Forms. The activity included multiple-choice quiz questions related to phishing attacks, suspicious links, password security, multi-factor authentication (MFA), and public Wi-Fi risks.

The purpose of the activity was to educate participants about common cybersecurity threats and improve their understanding of safe online practices.

## Implementation
I created an interactive quiz consisting of several cybersecurity awareness questions. The quiz tested participants on:

- Identifying phishing websites
- Recognising suspicious email behaviour
- Understanding MFA
- Password security
- Public Wi-Fi risks
- Proper response to phishing attacks

The learning activity was implemented using Google Forms to make the quiz easy to access and complete.

## Findings
The activity demonstrated that phishing awareness and password security are important cybersecurity concepts that many users still struggle with. The quiz helped reinforce safe online behaviour and encouraged participants to think carefully before clicking links or sharing credentials online.

## Reflection
This activity improved my understanding of cybersecurity education and awareness training. I learned that creating interactive learning activities can be an effective way to teach cybersecurity concepts in a simple and engaging manner. It also showed how awareness training can help reduce human-related security risks such as phishing attacks and poor password practices.
